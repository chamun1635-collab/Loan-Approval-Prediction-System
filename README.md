# Loan Approval Prediction System

## Overview
This project builds a machine learning model to automate loan approval decisions based on customer financial and personal data. The goal is to improve efficiency and reduce manual processing in loan approval systems.

## Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection and encoding
- Machine Learning model building
- Prediction of loan approval status

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Model
- Logistic Regression / Decision Tree / Random Forest (update based on what you used)
- Model trained on historical loan data

## Results
- Achieved good accuracy in predicting loan approval
- Important factors include income, credit history, and loan amount

## Dataset
- Public loan prediction dataset
